### 주간 스터디 루틴 (PostgreSQL + VS Code + GitHub)

#### 1. 평일: 개인 과제 (자율)

- **문제 확인:** GitHub `main` 브랜치에 올라온 문제를 확인
- **브랜치 생성:** VS Code에서 본인 이름 브랜치 만들기 (`git checkout -b feature/이름`).
- **풀이 및 정리:** `.ipynb` 파일에 SQL 쿼리와 실행 결과(Table)를 남기기 (필요시 태블로 캡처 포함)
- **제출:** 금요일 밤까지 `Push` 후 GitHub에서 **PR(Pull Request)을 생성**
